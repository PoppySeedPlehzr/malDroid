About malDroid

malDroid is an open source Andoird APK malware analysis engine, designed to aid in forensic and malware anlaysts. Originally developed as a class project for NYU: Polytechnic School of Engineering's Mobile Security course during Fall of 2014, this project hopes to grow into a framework that can be used by Malware Researchers, Incident Responders, and Academics, excited about malware reserach.
