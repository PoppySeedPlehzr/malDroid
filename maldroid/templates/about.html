About malDroid

malDroid is an open source Andoird APK malware scanner, designed to aid in forensic analysis. Originally devloped as a class project for NYU Polytechnic School of Engineering's Mobile Security course during Fall of 2014, this project hopes to grow into a framework that can be used by Malware Researchers, Incident Responders, and Academcis, excited about malware reserach.
